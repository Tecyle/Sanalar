字符分布统计
------------

本工具主要目的是通过分析大量样本文本来得出某一种固定语言的最常使用的单字分布，或者双字分布的分布概率。从而为编码猜测模块提供数据支持。
程序使用过程如下：

1. 使用 Language 参数指定程序运行；
2. 程序会在工作目录下搜索指定目录指定语言下的文件夹，扫描所有 txt 文件，对其进行分析；
3. 程序将分析结果以文本文件的形式写入指定文件。

字符分布
========

在单字符分布统计的时候，将忽略 ASCII 编码集。
运行逻辑如下：

* 统计总共字符个数；
* 统计空白字符个数；
* 统计每个字符出现的次数；
* 将每个字符出现的次数进行降序排序；
* 计算前 n 个字符的覆盖百分比。

双字符分布
=========

在双字符分布统计的过程中，将忽略多字节编码字符，即只针对 ASCII 编码集。
目前，程序逻辑主要面向中文，此部分实现暂缓。
